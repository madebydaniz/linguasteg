# linguasteg-models

Prototype language packs and gateway adapters for LinguaSteg.

## What this crate provides

- Prototype language packs for:
  - Farsi (`fa`)
  - English (`en`)
  - German (`de`)
  - Italian (`it`)
- Symbolic mappers, realizers, and text extractors for supported languages
- In-memory gateway registry and stub gateway/model adapter for local and CI workflows

## Intended users

Use `linguasteg-models` when you need ready-to-use prototype implementations on top of `linguasteg-core`.

## Notes

This crate focuses on deterministic prototype behavior and integration scaffolding. Language quality and stylistic realism are improved incrementally across releases.
