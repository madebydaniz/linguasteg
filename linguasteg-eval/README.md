# linguasteg-eval

Evaluation and metrics utilities for LinguaSteg.

## What this crate provides

`linguasteg-eval` contains evaluation-facing data structures and utilities used to score:
- Semantic similarity
- Payload accuracy
- Detectability risk

## Current status

This crate is intentionally minimal in the current release and is designed to grow as benchmark and regression workflows mature.
